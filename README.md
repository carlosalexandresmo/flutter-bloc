## Getting Started
I am assuming you have Flutter already setup.
For more information on how to setup Flutter, [click this link.](https://flutter.io/docs/get-started/install)
#### 1. Clone the Repo
```
$ git clone https://github.com/trinqk/flutter-bloc.git
$ cd flutter-bloc/
```
#### 2. Run the App
```
flutter run
```
